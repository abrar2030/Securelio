About Me

I am a motivated Cybersecurity and Cloud Security Engineer with over a year of hands-on experience in securing cloud-native infrastructures, automating security workflows, and implementing best practices to protect cloud environments. My expertise spans Kubernetes security, CI/CD pipeline hardening, container security, and infrastructure-as-code (IaC) security using Terraform and Helm. I specialize in securing cloud workloads, enhancing identity and access management (IAM), and implementing proactive monitoring solutions to ensure robust security, compliance, and high availability in modern IT ecosystems. I am now seeking a Junior to Medior Cloud Security or Cybersecurity Engineer role where I can contribute my skills while further deepening my expertise in cloud security, threat detection, and compliance automation.

At Ericsson, I worked extensively with Kubernetes, Docker, Helm, and CI/CD pipelines, focusing not only on cloud-native deployments but also on security optimization. I automated security checks in CI/CD pipelines using GitHub Actions and Jenkins, integrating vulnerability scanning tools such as Trivy and Clair to detect misconfigurations and threats in containerized applications. My role also involved enforcing Kubernetes security policies, implementing RBAC, network segmentation, and securing clusters with best practices like Pod Security Standards and runtime protection mechanisms.

Through project-based work, I have gained hands-on experience with AWS security, implementing IAM policies, security groups, and encryption strategies to protect cloud workloads. I have successfully deployed cloud security solutions on AWS EKS, automated infrastructure security compliance using Terraform and AWS Config, and designed monitoring solutions with Prometheus, Grafana, and AWS Security Hub. My experience in cloud-native security extends to workload protection, incident response, and compliance frameworks like CIS benchmarks, NIST, and ISO 27001.

I am passionate about securing cloud environments by integrating DevSecOps principles, automating security workflows, and ensuring compliance with industry standards. My strong technical foundation in security automation, identity management, and cloud networking enables me to design secure, scalable architectures that mitigate risks and enhance system resilience. I continuously explore advanced threat detection techniques, zero-trust architectures, and AI-driven security analytics to stay ahead in the rapidly evolving cybersecurity landscape.

With a degree in Computer Science from Eötvös Loránd University and practical experience in cloud security automation, I am eager to take on challenges that push the boundaries of cybersecurity innovation. My goal is to drive security-first cloud strategies, implement proactive defense mechanisms, and contribute to the development of secure, resilient infrastructures. I am actively seeking opportunities where I can apply my expertise in cloud security, DevSecOps, and threat intelligence while continuing to grow in a dynamic cybersecurity environment.
